# 文案產出規則與流程

## 📅 定時任務
- **時間**：每日早上 08:00。
- **頻率**：每個客戶每天各 2-3 篇文案。
- **目標客戶**：
  1. Dr.陳保中《健康減重，腸胃保安康》/ 保安診所
  2. 陳宣位醫師的健康日記
  3. 正農黑米，真正好農夫

## 📝 產出規範
- **文字風格**：遵循 SOUL.md（簡潔、有觀點、直言、無客套、機智），且**絕對禁止使用 Emoji**。
- **文章長度**：每篇文案**至少 600 字**，內容要深入且有厚度。
- **配圖建議**：每篇文案需附帶「配圖文字敘述」（傑哥自選圖）。
- **存檔路徑**：`C:\Users\User\Desktop\FB發文整理\`。
- **資料夾命名規範**：`客戶名\日期\文案標題\`。
- **內容規範**：資料夾內包含「文案內容」與「找圖文字敘述」。

## 🛠️ 執行流程
1. 每日 08:00 啟動文案撰寫作業。
2. 自動在桌面「FB發文整理」建立對應資料夾。
3. 寫入檔案並確認無亂碼（UTF-8 編碼）。
